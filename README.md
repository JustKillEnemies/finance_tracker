# finance_tracker
Путь проекта с нуля описан на страничке Wiki

В целях практики умений писать проекты на джанго, будет существовать данный финансовый веб-трекер, в котором можно следить за своими доходами, расходами, редактировать их, добавлять собственные категории расходов, изучать свои потребности, глядя на визуализированные отчеты по тратам
